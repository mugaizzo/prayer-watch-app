# Prayer Watch App

A Pixel Watch (Wear OS) application that displays Islamic prayer times using the Aladhan API.

## Features

- Display all five daily prayer times (Fajr, Dhuhr, Asr, Maghrib, Isha)
- Countdown timer showing remaining time until next prayer
- Location-based prayer times using Aladhan API
- Clean, circular design optimized for Pixel Watch
